## 🎯 切换逻辑

### 镜像（3 个）
```
优先级 1: C 盘 mirror      ← 默认使用
优先级 2: C 盘 mirror-1    ← C 盘故障时切换
优先级 3: D 盘 mirror      ← C 盘双故障时启用（杀手锏）
```

### 备份（3 个）
```
优先级 1: C 盘 backup-1    ← 默认使用
优先级 2: C 盘 backup-2    ← C 盘故障时切换
优先级 3: D 盘 backup      ← C 盘双故障时启用（杀手锏）
```

## 📌 核心功能

1. **自动检测** - 每次运行自动检查 6 个目录健康状态
2. **自动切换** - 发现故障自动切换到备用目录
3. **自动恢复** - 主目录恢复后自动切回
4. **日志记录** - 所有切换事件详细记录
5. **状态持久化** - 状态保存至 JSON 文件

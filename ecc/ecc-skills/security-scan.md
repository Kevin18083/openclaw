# 安全扫描技能

> 来源：ECC security-scan + AgentShield 集成

---

## 触发词

```
安全扫描、security scan、检查漏洞、安全审计
```

---

## 扫描项目

### 1. 敏感数据
- [ ] 无 .env 文件提交
- [ ] 无硬编码 API Key
- [ ] 无硬编码密码
- [ ] 无硬编码 Token

### 2. 注入风险
- [ ] 无 SQL 注入
- [ ] 无命令注入
- [ ] 无 XSS 风险
- [ ] 无路径遍历

### 3. 依赖安全
- [ ] 无已知漏洞依赖
- [ ] 使用最新稳定版本
- [ ] 无废弃包

### 4. 配置安全
- [ ] CORS 配置正确
- [ ] 错误信息不泄露
- [ ] 生产环境调试关闭

## 扫描命令

```bash
# 运行安全扫描
node malware-detector.js

# 深度扫描
node malware-detector.js --deep

# 检查依赖
npm audit
```

---

## 风险等级

| 等级 | 颜色 | 说明 | 处理 |
|------|------|------|------|
| 严重 | 🔴 | 立即阻塞 | 必须修复 |
| 高危 | 🟠 | 尽快修复 | 24 小时内 |
| 中危 | 🟡 | 计划修复 | 本周内 |
| 低危 | 🟢 | 可选修复 | 记录即可 |

## 与杰克系统集成

1. **提交代码** → 扎克提交修改
2. **触发扫描** → "让杰克检查安全"
3. **AI 分析** → DeepSeek 分析扫描结果
4. **输出报告** → 风险等级 + 修复建议
5. **扎克修复** → 按建议修复问题
6. **重新扫描** → 验证修复

## 命令集成

```bash
# 创建安全检查任务

# 提交扫描结果

# 查看状态
node jack-review.js status
```

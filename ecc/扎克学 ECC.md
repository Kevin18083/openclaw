## 🚀 扎克学习流程

### 第一步：理解 ECC 核心思想

1. **测试驱动** - 先写测试，再写代码
2. **代码审查** - 所有代码必须经过审查
3. **持续学习** - 每次任务后提取经验
4. **安全第一** - 敏感操作要检查

### 第二步：掌握常用命令

```bash
# 规划任务
/plan [任务描述]

# 测试驱动开发
/tdd [功能描述]

# 持续学习
/learn [主题]

# 代码审查
/code-review [文件/代码]
```

### 第三步：与杰克系统联动

```bash
# 创建检查任务
node jack-review.js "让杰克检查 [任务名]"

# 提交内容
node jack-review.js submit "任务详情"

# 查看状态
node jack-review.js status
```

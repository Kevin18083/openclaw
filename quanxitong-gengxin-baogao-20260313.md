### 1. 看门狗脚本
**文件**: `C:\Users\17589\.openclaw\zack-watchdog.js` (4.2KB)

**功能**:
- 每 10 秒检查扎克进程
- 扎克挂了 → 3 秒内自动重启
- 连续失败 3 次 → 发送钉钉告警

**配置**:
```javascript

restartDelay: 3000,        // 重启延迟 3 秒
maxFailures: 3,            // 告警阈值 3 次
```

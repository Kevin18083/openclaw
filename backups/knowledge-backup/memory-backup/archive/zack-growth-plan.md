## 教程 02: 安全编程实践 🔄

**状态**: 进行中（下一步创建）

**计划内容**:
- 敏感信息处理（密码、token、密钥）
- 输入验证与 sanitization
- 路径遍历防护
- SQL 注入防护
- XSS 防护基础
- 权限最小化原则
- 安全日志记录

**安全等级对照表**:
```
🔴 Critical - 立即修复（如明文存储密码）
🟠 High - 需要关注（如硬编码密钥）
🟡 Medium - 建议修复（如缺少输入验证）
🟢 Low - 可接受（如日志过于详细）
```

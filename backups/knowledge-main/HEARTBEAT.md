# HEARTBEAT.md - 主动代理检查清单

## 每日检查（2-4 次）
- [ ] 检查未读重要邮件
- [ ] 查看今日日历事件（未来 2 小时内）
- [ ] 检查待办事项优先级
- [ ] **监控系统资源状态**（performance-monitor.js，每 4 小时自动执行）
- [ ] **验证记忆系统完整性**（memory-maintenance.js）
- [ ] **运行阿里云 API 健康检查**（每小时自动执行）
- [ ] **检查错误日志**（error-log.md）
- [ ] **检查备份状态**（auto-backup.js，每日 00:00 自动执行）

## 每周检查
- [ ] 回顾本周完成的任务
- [ ] 计划下周重点工作
- [ ] 检查项目进度
- [ ] 清理临时文件
- [ ] **深度整理长期记忆**（新增）

## 记忆维护规则
- **每日 00:00**：自动运行记忆系统维护（memory-maintenance.js）
- **每日 18:00**：记忆更新（cron 任务）
- **每次心跳**：验证记忆文件存在性
- **学习完成**：立即保存到记忆文件
- **重要对话**：自动记录到日常记忆

## 提醒规则
- 工作时间（9:00-18:00）：正常提醒
- 休息时间（18:00-9:00）：仅紧急提醒
- 周末：仅重要事项提醒

## 检查频率
- **心跳检查：每 4 小时一次** ✅（已调整）
- 重要检查：每 4 小时一次
- 常规检查：每天 6 次（工作时间 9:00-21:00）
- 深度检查：每周一次（周一 10:00）

## 系统脚本清单
- `memory/memory-maintenance.js` - 记忆维护（每日 00:00）
- `scripts/auto-backup.js` - 自动备份（每日 00:00）
- `scripts/performance-monitor.js` - 性能监控（每 4 小时）
- `error-log.md` - 错误日志
- `self-optimization-log.md` - 自我优化日志
- `file-lookup-table.json` - 快速查找表

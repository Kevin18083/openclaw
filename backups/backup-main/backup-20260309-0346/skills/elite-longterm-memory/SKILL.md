# Elite Longterm Memory Skill

## 功能特性

### 1. 记忆分类系统
- **短期记忆**：临时存储，24小时后自动归档
- **中期记忆**：重要信息，保留7-30天
- **长期记忆**：核心知识，永久保存
- **归档记忆**：历史记录，按需检索

### 2. 智能记忆管理
- 自动分类和标记
- 基于重要性的优先级排序
- 相似记忆合并与去重
- 过期记忆自动清理

### 3. 高级检索功能
- 语义搜索（基于内容含义）
- 时间线浏览（按时间顺序）
- 标签过滤（按分类筛选）
- 相关性排序（智能推荐）

### 4. 记忆优化
- 定期记忆整理
- 重要性重新评估
- 存储空间优化
- 备份与恢复

## 文件结构

```
memory/
├── short-term/      # 短期记忆（24小时）
├── medium-term/     # 中期记忆（7-30天）
├── long-term/       # 长期记忆（永久）
├── archived/        # 归档记忆
├── tags/           # 标签索引
└── indexes/        # 搜索索引
```

## 基本配置

### 1. 初始化记忆系统
```bash
# 创建记忆目录结构
mkdir -p memory/{short-term,medium-term,long-term,archived,tags,indexes}
```

### 2. 设置记忆策略
创建 `memory/config.json`：
```json
{
  "retention_policies": {
    "short_term": "24h",
    "medium_term": "30d",
    "long_term": "permanent"
  },
  "auto_cleanup": true,
  "backup_enabled": true,
  "compression_enabled": true
}
```

### 3. 日常维护任务
```bash
# 每天凌晨3点进行记忆整理
openclaw cron add --schedule '0 3 * * *' --message "执行每日记忆维护：分类、清理、优化"
```

## 使用示例

### 1. 添加新记忆
```bash

echo "项目关键决策：采用微服务架构" > memory/long-term/project-architecture.md

# 添加日常记忆（自动分类为中期记忆）
echo "会议记录：讨论了Q2目标" > memory/medium-term/meeting-2026-03-04.md
```

### 2. 搜索记忆
```bash
# 语义搜索
grep -r "微服务" memory/

# 时间范围搜索
find memory/ -name "*.md" -newermt "2026-03-01" ! -newermt "2026-03-05"
```

### 3. 记忆整理
```bash
# 移动过期记忆到归档

# 重新评估记忆重要性
# （基于访问频率、关联性等指标）
```

### 4. 生成记忆报告

## 故障排除

### 常见问题
1. **记忆找不到**：检查搜索索引，运行 `重建索引` 命令
2. **存储空间不足**：启用压缩，清理过期记忆
3. **性能下降**：优化索引，减少实时搜索范围

# 杰克血泪史 — Bug 教训记录

> **原则**：同一个错误，绝不犯第二次！
>
> 每次修复 bug 后，记录到这里。每次写代码前，先看一遍！

---

## 📖 使用说明

**每次修复 bug 后填写**：

```markdown
### Bug #XXX - [简短描述]

**日期**: YYYY-MM-DD

**现象**: [用户报告了什么]

**根因**: [真正的根本原因，不是表面]

**修复**: [怎么修的]

**预防**: [以后怎么避免]

**教训**: [一句话总结，贴在脑门上]
```

**每次写代码前，重读一遍这些教训！**

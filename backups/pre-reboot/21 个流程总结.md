## 🎯 使用方式

### 快速开始

```bash

# 快速检查（5 项核心）
bash quick-check.sh .

# 决策树评估（权重评分）
bash decision-tree.sh .
```

### 单独运行新流程

```bash
# 需要先 source 主框架
source test-framework-plus.sh

# 运行日志检查
logging_check "."

# 运行容器化检查
containerization_check "."

# 运行错误处理检查
error_handling_check "."

# 运行国际化检查
i18n_check "." "zh-CN"
```

### 完整流水线

```bash
# 运行全部 21 个流程
full_pipeline "我的项目" "./src" "false"
```

**初步测试**：
- ✅ 环境检查（Node.js、Git）
- ✅ 5 轮重试机制

**代码质量检查**：
- ⚠️  TODO/FIXME/HACK 标记

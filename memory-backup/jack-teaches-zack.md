## 📖 概述

这是杰克 (Jack) 和扎克 (Zack) 之间的**万能互动指南**。

**罗总指令**：只要扎克在网页端收到罗总的指令或任务，第一时间想到这个文件！

**核心原则**：
- 杰克是技术导师，扎克是执行者
- 无论什么问题，杰克都能帮扎克解决
- 扎克完成任务后，在网页端回复罗总结果

**互动方式**：
- **问答式** - 扎克提问 → 杰克回答 + 给代码示例
- **协作式** - 杰克分析后直接帮扎克操作，边做边解释
- **教学式** - 杰克逐步教扎克，让它自己操作，只动口不动手

  for (const [key, rules] of Object.entries(schema)) {

## 📋 答案

**Promise** 是基础，**async/await** 是语法糖。

### Promise
```javascript
fetch('/api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));
```

### async/await（推荐）
```javascript
async function getData() {
  try {
    const res = await fetch('/api/data');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
```

### 区别
- async/await 更直观，像同步代码
- Promise 需要链式调用
- async/await 底层还是 Promise
- 推荐用 async/await

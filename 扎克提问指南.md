## 🎯 扎克，遇到问题这样做：

### 第 1 步：先查指南
打开 `扎克完全指南.md` 查找相关章节。

### 第 2 步：运行诊断命令
```bash
# 查看检查状态
node jack-review.js status

# 查看缓存状态
node zach-cache-optimizer.js status

# 查看备份状态
node 3plus3-auto-switcher.js status
```

### 第 3 步：提交杰克检查任务（推荐）
```bash
node jack-review.js "让杰克检查 [你的问题]"
```

### 第 4 步：等待杰克自动解答
杰克会在 1 秒内生成答案并推送到控制台！

## 📋 提问模板

**复制这个模板，填空即可：**

```bash
node jack-review.js "让杰克检查 [问题描述]"
```

**示例：**

| 扎克输入 | 杰克回答 |
|----------|---------|

| `node jack-review.js "让杰克检查 缓存命中率上不去"` | 推送缓存优化方案 |
| `node jack-review.js "让杰克检查 这个报错怎么办"` | 推送错误解决方案 |

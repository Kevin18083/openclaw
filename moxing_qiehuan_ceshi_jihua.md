# 模型自动切换系统 - 测试计划

## 测试目标
验证保守切换策略的稳定性和可靠性，确保不会误切换，同时在真实故障时能及时响应。

## 测试配置

### TEST 版本参数
```javascript
{
  failureThreshold: 3,      // 连续失败 3 次才切换（原：1 次）
  checkInterval: 30000,     // 每 30 秒检查一次
  requestTimeout: 15000,    // 15 秒超时
  autoSwitchEnabled: true,  // 启用自动切换
  recoveryThreshold: 5      // 连续成功 5 次才切回（新增）
}
```

### 对比：原版本 vs TEST 版本
| 参数 | 原版本 | TEST 版本 | 改进 |
|------|--------|----------|------|
| 失败阈值 | 1 次 | 3 次 | 避免瞬时故障误切换 |
| 恢复阈值 | 无 | 5 次 | 确保服务真正稳定 |
| 超时时间 | 10 秒 | 15 秒 | 更宽容网络波动 |
